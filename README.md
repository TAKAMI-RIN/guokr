#果壳前端组件

##目录结构

    ./
      build/        构建工具
      example/      javascript组件样例
      js/           javascript目录
          G/            G.js库
      skin/         css及图片目录

##如何开始
在当前目录启动一个server，然后访问[http://127.0.0.1/](http://127.0.0.1/)。
或者安装python，然后运行http.sh，访问[http://127.0.0.1:8000/](http://127.0.0.1:8000/)。
