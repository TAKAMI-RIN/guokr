简易tab

描述：
* 简易的tab模块
* @author mzhou
* @version 1.2
* @description 真的很简易
* 根据trigger和target一一对应的切换
* @log 1.1 添加了targetSelector,与前一版本不再兼容
* 1.2 添加了callback回调，参数为currentTrigger（当前tab的jquery对象）、currentPanel（当前panel的jquery对象）
引用js和css：
JS:
//@import "lib/tab.js";
API接口
//profile.js
/**
 * return function( triggerPanel, triggerSelector, targetPanel, targetSelector, currentClass, callback ) 
 */
    // tab切换
    var tabber = tab( '#switcherTitle', 'a', '#switcherContent', 'div,ul', 'current', function ( $panel, $trigger ) {
        // 此处可以设置：用户本人的页面时，显示提醒“设置我关注的人”等
        var text = $trigger.text();
        $('#currentPanel').text(text);
        return true;
    }),
        t =  3;         //选中的链接，此处为“关注的人”
    tabber.set( t-1 );
