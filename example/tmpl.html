简易模板tmpl

描述：
* 基于john resig的micro template写的前端模板模块
* @author mzhou
* @version 1.0
* @description 此模块基于自john resig的micro template，最大的特色是代码很少，性能处于中等水准<http://jsperf.com/dom-vs-innerhtml-based-templating/73>
引用js和css：
JS:
//@import "lib/tmpl.js";
API接口
    var $image = $('#showImage'),
    // demo中 替换的文本
    path = 'http://www.guokr.com/gkimage/16/c0/6l/16c06l.png',
    // demo中 id名为 tp_image
    tp = tmpl('tp_image');

    $image.empty().html( tp({ src: path}));
